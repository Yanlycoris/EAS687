EAS687 3D modeling Tutorial - SU

Yan Li 03/20/2021

This tutorial aims to go through how to use Sketchup to create a 3D terrain model. We are going to use a built-in function called "Geo-location" to download the terrain data and use model-intersection to create the terrain base ( make it more like a 3D physical cutting of earth). I will also show you how to paint the materials and add more components to modify your model.

Let's start it!

Advantages of SketchUp:

  • Easy to use
  • Build Small-scale 3D Terrains in 5 steps
  • Easy to add trees/people/building

Download SketchUp Pro from its official website here .

Open SketchUp Pro.

Click File>Geo-location>Add Location...

Sign-in (It's free)

Search the address.

select the region (available when zoom level>14)

Choice Provider(some version may skip this step - it's fine)

click import.

Now you are import the satellite map to SU.

Navigate to File>Geo-location>Show Terrain. After click, the Terrain surface will show up.

#Checkpoint 1 - Generate Terrain Surface

You can use the Tape measure tool to measure the dimension. (the number will be shown in the right corner also). Measure the height of the terrain, and write down this number.

Next, we are going to make the terrain base.

Add tag (similar to layer in GIS).

Name it :)

Make the new tag as your working one.

Navigate to Camera>Standard Views>Top.

Click the ‚Äúeye‚Äù icon to hide the terrain layer.

Use shapes tool draw a rectangular surface. (Note: slightly similar than the image)

Change camera to perspective.

Open the terrain layer.

Click the surface, then use the push/pull tool to make a cuboid. Make sure the height is lager than the terrain's elevation range.

Hide image and terrain layer. Select the 3D box, and right click and select make group.

Use move tool to move the box, make sure the box is smaller than the terrain surface.

(you can use the mouse middle to change the camera at the same time). We are going to use the intersect the terrain and box model to get a 3D terrain.

If your box is two large or not high enough, use scale tool to resize.

Be sure to make sure that the box can be completely cut on the four sides of the terrain and the Z-axis is wrapped in a cuboid. We can also leave the LOGO outside of the box to cut it later.

Error cases:

Press Ctrl+S to save your model. (In case failure...)

#Checkpoint 2 PREPARE TO INTERSECT MODEL

Select all, right click Unlock first, then press ctrl+C and ctrl+V to make another copy.

(The additional copy is used for redo - in case of failure. We will still work on the original one)

Right click explore.

Reselect all. Right click Intersect Faces>With Model.

Then you can find the box cut by the Terrain surface.

Delete the top surface.

Hold ctrl to select all the faces we want, then right click to make group.

#Checkpoint 3 - 3D MODEL

(If you find errors, carefully check the reasons. If it not fully cut, use another copy to repeat the steps of the intersection model. Don't forget to make another copy before restart!)

Right click the group, choose lock. the blue frame will change to red color.

Select all then press delete(Don't worry, our model is locked!). In this way, we can clean the unwanted debris quicker.

We almost finish our model. Yay!

Let's do the last modify - change the material of the side faces. Make it a little bit more elegant.

Right click to choose unlock the model, then right click again to edit the group.

Navigate to the material tab on the right

Choose the material you like, and use the paint bucket tool to paint the material to the faces.

You can also edit the materials or import the pattern you like.

You can also open shadows by navigate to View>Shadow. and use the right Shadow tab to adjust.

Navigate to View>Axes, uncheck it to hide the spatial axes.

#Checkpoint 4 - 3D TERRAIN MODEL

Congratulations!!!

#Checkpoint 5 - Have fun!

Add trees/people/structures...

Unfold the right tab of Components, type the components in the search bar. You can also create your own model in SketchUp :)

Click the tree you want.
